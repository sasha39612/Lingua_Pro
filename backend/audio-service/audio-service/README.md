# Audio Service

This is the audio service for the Lingua project, built using NestJS. 

## Overview

The audio service is responsible for handling audio-related functionalities within the application. It serves as a backend service that can be integrated with other components of the Lingua project.

## Installation

To install the necessary dependencies, run:

```
npm install
```

## Running the Application

To start the audio service, use the following command:

```
npm run start
```

The application will listen on the port specified in the `.env` file or default to port 4003.

## Environment Variables

Make sure to create a `.env` file in the root of the project with the following variable:

```
PORT=4003
```
